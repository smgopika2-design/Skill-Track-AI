## 📖 Overview

SkillTrack AI is an AI-powered Machine Learning project developed to predict **skill retention** and **knowledge decay** among students. The system analyzes learning behavior, quiz performance, revision frequency, and engagement patterns to estimate the likelihood of forgetting a skill and provides personalized recommendations to improve long-term retention.

This project was developed as part of the **BCA (Data Science & Artificial Intelligence) Internship Project**.

---

## 🎯 Problem Statement

Students continuously acquire new skills through courses, certifications, and self-learning. However, without regular revision and practice, these skills gradually decline over time. Existing learning platforms primarily focus on learning new content rather than helping users retain previously acquired knowledge.

SkillTrack AI addresses this challenge by predicting skill decay and recommending timely interventions to improve knowledge retention.

- ML Algorithm: Random Forest
